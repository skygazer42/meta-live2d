## 嵌入个人网站
### iframe方式
1. 将[iframe.html](./iframe.html)中的`iframe`部分替换为[skygazer42](https://github.com/skygazer42)分享应用中`iframe`的代码。
2. 使用浏览器打开[iframe.html](./iframe.html)即可。
### script方式
1. 将[script.html](./script.html)中的`script`部分替换为[skygazer42](https://github.com/skygazer42)分享应用中`script`的代码。
2. 使用浏览器打开[script.html](./script.html)即可。
